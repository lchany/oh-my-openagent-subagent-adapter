# OpenCode architecture

This directory is the independent OpenCode implementation. It contains:

- `versions/v1/patches/`: OpenCode adapter and workflow activation patches;
- `docs/`: OpenCode patch background, implementation, and application instructions;
- `verl-subagent-union-workflow/`: OpenCode agents, contracts, scripts, tests, plugins, and workflow skill.

Install OpenCode agents and skills only from `verl-subagent-union-workflow/`. Load only files owned by this directory.

The OpenCode workflow package has its own [README](verl-subagent-union-workflow/README.md) and [manifest](verl-subagent-union-workflow/manifest.md).
